# Installation Instructions
# EN.705.743.8VL.SP26 — KV-Cache / Speculative Decoding Project

## Requirements

- Python >= 3.10
- PyTorch >= 2.0  
- NumPy >= 1.24
- Matplotlib >= 3.7

## Installation

### Option A — pip 

```bash
pip install torch numpy matplotlib
```

### Option B — conda

```bash
conda create -n kvcache python=3.11
conda activate kvcache
conda install pytorch numpy matplotlib -c pytorch
```

## Dataset

Tiny Shakespeare is downloaded:

```bash
python main.py  # downloads to ./data/shakespeare.txt
```

No manual download needed.

## Running

```bash
# Full run (trains + benchmarks + demo)
python main.py

# Quick test (everything runs in under 5 minutes on CPU)
python main.py --quick

# If models are already trained
python main.py --no-train
```
