  at every position, thus identical output token sequences.

Experiment 3: Complexity Analysis
======================================================================
  Baseline scales approximately as O(n²):
  Each new token requires a full forward pass over the entire sequence.
  Attention cost grows quadratically with context length.

  KV-Cache scales approximately as O(n) per token:
  Only Q,K,V of the single new token are computed; past K,V are read
  from cache. Attention with all cached keys is O(T) per step.

  Speculative Decoding reduces number of main-model forward passes:
  K draft tokens verified in one pass → fewer expensive calls needed.
